---
title: vue 过滤器(filter)

---



**什么是: 接收原始值，进行再加工**

 **何时: 只要原始值无法直接使用时**

 **比如: 性别: 0，1 ——> 女  男**



  **inserted(el){ //会在当前元素被添加到DOM树上时自动调用**

   **//el: 当前DOM元素**

   **//可对el执行原生DOM API**

​    **}**

 **如何:**

**1. 创建:**

```
 Vue.filter(“过滤器名”, function(val){
   return 加工后的新值
})
```



**2. 使用自定义过滤器:**

```
 在{{}}中使用: {{模型变量 | 过滤器 | … }}
 在属性绑定时使用:  :属性名=”表达式 | 过滤器 | … ”
 v-text不能用过滤器！
错位传参（从形参第二位开始传参）
```
