## 1.使用

```
在创建项目的时候 使用ESLint 然后使用分开保存 就会生成ESLint文件
```

## 2.创建原生项目

```
1. npm init -y  //创建package.json 管理第三方依赖包
2. npm add -D eslint  //安装eslint
3. npx eslint //找到node_modules eslint 文件
4. npx eslint --init 
   选择 检查语法 发现错误  To check syntax and find problems
   选择 项目中的风格  JavaScript modules(import/export)
   选择 你是什么项目 Node of these
   选择 运行环境 Browser Node 两个运行环境
   选择 你是希望以什么方式做配置 Javascript
```
